******************************************
MIVisionX tests
******************************************

MIVisionX includes a test suite to verify installation and functionality. 

* `OpenVX 1.3 Conformance tests <https://github.com/ROCm/MIVisionX/tree/master/tests/conformance_tests>`_ for Vision Feature Set for CPU and GPU (OpenCL & HIP Backend)
* `Library tests <https://github.com/ROCm/MIVisionX/tree/master/tests/library_tests>`_ to check if all libraries are built and installed
* `Neural Network tests <https://github.com/ROCm/MIVisionX/tree/master/tests/neural_network_tests>`_ to verify function and performance of Caffe, ONNX, and NNEF model flows with OpenVX
* `OpenVX Node tests <https://github.com/ROCm/MIVisionX/tree/master/tests/openvx_node_tests>`_ for AMD OpenVX functionalities in HOST and HIP and OpenCL backends
* `Smoke tests <https://github.com/ROCm/MIVisionX/tree/master/tests/smoke_tests>`_ for quick MIVisionX tests
* `Vision function tests <https://github.com/ROCm/MIVisionX/tree/master/tests/vision_tests>`_ for verification and performance
* `Zen DNN Unit tests <https://github.com/ROCm/MIVisionX/tree/master/tests/zen_dnn_tests>`_ for unit test, verification and performance
